# Assets Directory

Add your image and sound files here. The app expects the following:

## Required for app to run

| File | Used in |
|------|--------|
| `logo.png.jpg` | Login page |
| `image2.webp` | Child/parent details page |
| `jungle_scene.jpg` | Tap the Sound game (background) |

## Tap the Sound game – animals (images + sounds)

| Image file | Sound file |
|------------|------------|
| `bird.png.jpg` | `bird sound.mp3.wav` |
| `cat.png.jpg` | `cat sound.mp3.wav` |
| `dog.png.png` | `dog sound.mp3.wav` |
| `cow.jpg` | `cow sound.mp3.wav` |
| `rooster.png.jpg` | `rooster sound.mp3.wav` or `rooster sound.mp3` |

## Optional

- `tap the animal sound.mp3.mp3` – title sound for Tap the Sound game  
- `happy-success.mp3` – correct-answer feedback  
- `sad-incorrect.mp3` – wrong-answer feedback  

---

**Note:** No asset files were moved to the script folder. If these files are missing, add them from your backup or replace them with your own images/sounds and update the paths in the code if needed.
